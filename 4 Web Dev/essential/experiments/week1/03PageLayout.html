About this experiment

This experiment demostrates simple HTML page layout. This page uses HTML and CSS. No inline CSS is used here. All CSS are defined in separate file for reusability. In these experiments, I have tried to design a simple layout of a webpage. At the top there is header. On left hand side, navigation links are present. The main contents of the page are at the center. The footer contents are bottom

This is achieved by using div in HTML and then mapping each div in css for formatting. Each div is identified with an id.

Code snippet

HTML div

        <div id="header"> 
             <h1> Here goes Header !!!  </h1> 
        </div>

CSS 

        #header
        {
        background-color: black;
        color: white;
        text-align: center;
        padding: 5px;
        }
