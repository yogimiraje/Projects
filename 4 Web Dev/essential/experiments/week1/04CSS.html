Please note that this may not work in Internet Explorer. Try Chrome/Safari.

About this experiment

In this experiment, some of the CSS features are demonstrated. The background image is set. Horizontal navigation bar is created. Also, the color of visited link is changed. Text effect is shadow is shown. An animationis created where colors of square block change for first 10 seconds.

Code Snippet

     CSS code for animation: 

    .anim {
        width: 100px;
        height: 100px;
        background: red;
        -webkit-animation: myfirst 10s;
        animation: myfirst 10s;
        }

     @-webkit-keyframes myfirst {
     from {background: red;}
     to {background: yellow;}
     }

    @keyframes myfirst {
    from {background: red;}
    to {background: yellow;}
    }
