About this experiment

In this experiment, I have tried to make the page responsive. So, irrespective of the client device you will use, you can see the contents without having to scroll horiontally.
On laptop or desktop, you will see the text with grey background. When you resize the window to tablet-width, you can see the background color changing to red. Try resizing window to mobile width and you will see the background color becomes yellow. Thus, the page response is media-specific. This is achieved by using @ media screen feature of CSS and making the reponse predictable.

Code Snippet

CSS code

 
        @media screen and (max-width:350px)
        {
         body{ background-color:yellow;
        }
        }
        @media screen and (min-width:350px) and (max-width:800px)
        {
         body{
         background-color:red;
        }
        }
        @media screen and (min-width:800px)
        {
         body{
         background-color:grey;
        }
        }
