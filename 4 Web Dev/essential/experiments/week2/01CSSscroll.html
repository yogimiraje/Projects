About this experiment

This experiment demostrates text box with scoll bar. Sometime the text size is more than the box in which it should fit.In this case, we have to implement a text box with scroller, so all the text can be read.

This is achieved by using div scroll.

Code snippet

HTML div

        <div class="scroll"> text here......</div>

CSS 

     div.scroll 
       {
        background-color: #00FFFF;
        width: 200px;
        height: 200px;
        overflow: scroll;
        }
