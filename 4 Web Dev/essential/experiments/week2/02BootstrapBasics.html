2.2 Boostrap Basics

About this experiment

As you may have noticed, this page is much more elegant than the previous boring and dull page that was implemented in CSS. Thanks to default styling of Bootstrap !

This experiment demonstrates some of the basic features provided by boostrap. Boostrap class navbar is very powerful when it comes to navigation. You may see the hover effect, responsiveness of the navigation. As you reduce the size of this page at one point, you will see that the navigation bar is replaced by hold-down element that gives the navigation menu on clicking. The "form-control" class is used for search box.

Code snippet

HTML:

        <nav class="navbar navbar-inverse navbar-fixed-top">
            <div class="container">
               <div class="navbar-header">
                  <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#navbar" aria-expanded="false" aria-controls="navbar">
                     <span class="sr-only">Toggle navigation

                     <span class="icon-bar"></span>
                     <span class="icon-bar"></span>
                     </button>
                 <a class="navbar-brand" href="#">Product Review
              </div>
         </div>
