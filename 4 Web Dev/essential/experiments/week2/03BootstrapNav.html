About this experiment

This experiment illustrates the some advanced features of navigation bar such as drop down, search box embedding. "dropdown" class of boostrap is used here. It also show how to design table using bootstrap. The table is implemented with "table table-hover". It has nice hover effect. The colors are bootstrap standard colors in order - active,success,danger,info,warning.

HTML

        <li class="dropdown">
            <a href="#" class="dropdown-toggle" data-toggle="dropdown" role="button" aria-expanded="false">Categories <span class="caret"></span></a>
                 <ul class="dropdown-menu" role="menu">
                    <li><a href="#">Appliance</a></li>
