2.4 Boostrap Jumbotron

About this experiment

This experiment demonstrates usage of jumbotron class. This is a template for a product review website. It includes a large image spread at the center something that is catchy. Usually this is a moving image / flash. Just below that there are three supporting pieces of information. This makes a perfect home page for some infomational website !
