3.2 JavaScript onmousehover Image

Mouse over the sun and the planets and see the different descriptions.

About this experiment

This experiment demonstrates how you can use javascript onmousehover feacture to display the text your mouse pointing to. In the image, just hover over the planets to find more about that planet. Certain portion of image is marked with coordinates to make the boundries and whenever the mouse is hovering over that portion it displays the corresponding text.

Code snippet

             function writeText(txt) {
                document.getElementById("desc").innerHTML = txt;
             }
