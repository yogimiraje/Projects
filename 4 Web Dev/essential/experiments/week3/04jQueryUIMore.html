About this experiment

This experiment shows you how jQueryUi helps you resizing the images. The default resizing is only for three direction. When you add handles for the directions you want, you will get the resizing in desired directions. You may also restrict the extent to which you need resizing by specifying minWidth, minHeight, maxHeight, maxWidth.

Code snippet

            $(function () {
                $(".resizeme").resizable();
                $(".resizemeall").resizable({
                    handles: "n, e, s, w, ne, se, sw, nw"
                });
                $(".resizeme1").resizable({
                    minWidth: 200,minHeight:200,maxHeight:800,maxWidth:800
                });
            })
