About this experiment

This experiment introduces some simple things you can achieve using Angular JS. The first part allows dynamic update of the HTML. This achieved by same value of ng directives to paragraph and input form. In the second example, a jSON object is rendered on HTML page. For the sake of ease, I have used a hardcoded object, however in practical applications, object is retrieved from web API as we saw in previous examples.

Code snippet

HTML:

 

            <p> Type here: <input type="text" ng-model="text" />  </p>
            <p> You are typing.... </p>
            <p ng-bind="text"> </p>
        
            <tr ng-repeat="p in products">
            <td> {{p.pname}} </td>
            <td> {{p.price}} </td>
            </tr>

Angular JS:

            var app = angular.module("BasicApp", []);
            app.controller("BasicAppController",
            function ($scope) {
                var product = {
                    pname: "nexus",
                    price: "$200"
                };
                $scope.product = product;
                var products = [
                    { pname: "iphone", price: "$500" },
                    { pname: "ipad", price: "$700" },
                    { pname: "macbook pro", price: "$1000" },
                    { pname: "macbook pro retina ", price: "$1300" },
                    { pname: "iMac", price: "$2100" },
                ];
                $scope.products = products;
  
            });
