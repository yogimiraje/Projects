4.4 Angular JS Event Handling

About this experiment

This experiment explains the even handling feature of Angular JS. When you click on any of the images above, name is updated dynamically on the web page. This is achieved by "ng-click" directive that calls respective function in jQuery, that in turn updated the string object on the page.

Code snippet

HTML:

            <div class="col-sm-3">
                <img ng-click="empire()" src="../../images/empire.jpg" class="img-circle" width="150" height="200" />
            </div>

Angular JS:


             var app = angular.module("LocationApp", []);
                app.controller("LocationAppController",
                 function ($scope) 
                   {
                        $scope.LocationDetails = "New york !"
                        $scope.empire = function () {
                            console.log("Empire");
                            $scope.LocationDetails = "Empire State Building "
    
                    }
